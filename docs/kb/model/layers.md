# Two layers

**Documentary layer** — source-near markers the register itself supplies: date (with *more veneto* only where marked), folio extent, marginal note, text, subscription block, relations. **Analytical layer** — analyst-imposed vocabularies (policy_domain, decision_orientation, document_trigger, actors, geography): every value is a hypothesis with rule id, evidence span and validation status. The distinction is the method: sources with fields (cadastres, apprenticeship contracts) let extraction read structure off the page; the Risposte have none, so the structure is imposed and must carry its evidence.
